# Agent Skills

Reusable skill definitions for [OpenCode](https://opencode.ai). Each skill lives in its own folder with a `SKILL.md` file following the [skill spec](https://opencode.ai/docs/skills/).

## Skills

| Skill | Description |
|-------|-------------|
| `project-resume-evaluator` | Evaluates whether a project is resume-worthy. Scoring rubric, best practices audit (backend+fullstack), gap analysis, and next project suggestions. |
| `project-interview` | Mock technical interview about a project. MCQ and short-answer format with feedback after every response. Detects vibe coding. |
